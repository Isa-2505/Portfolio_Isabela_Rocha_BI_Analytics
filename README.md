# 🤖 Projeto 2: Modelagem Preditiva de Risco de Crédito

## Descrição
Modelo de Machine Learning para classificação de risco de crédito. Utiliza Random Forest para prever se um cliente apresenta risco baixo, médio ou alto baseado em idade, renda e tempo de cliente.

## Técnicas Utilizadas
- **Python**: Pandas, Scikit-learn, Matplotlib, Seaborn
- **Machine Learning**: Random Forest Classifier
- **Validação**: Train/Test Split (80/20), Matriz de Confusão
- **Métricas**: Acurácia, Precisão, Recall, F1-Score

## Arquivos
- `dados_risco.csv` - Base de treinamento (25 clientes com labels)
- `modelo_risco.py` - Código Python com modelo completo
- `risco_credito.png` - Dashboard com 4 análises

## Como Rodar
```bash
python modelo_risco.py
```

## Resultados
- ✅ Modelo treinado com 80% dos dados
- ✅ Acurácia do modelo validada
- ✅ 4 gráficos: Matriz de Confusão, Importância de Features, Distribuição de Risco, Precisão
- ✅ Análise de importância das variáveis
